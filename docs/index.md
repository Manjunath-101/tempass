# Rocket Shooter Game

This site hosts documentation for the Rocket Shooter game.

## How to play

- Use arrow keys to switch lanes.
- Press space to fire lasers.
- Collect coins and shoot aliens.

See the [README](../README.md) for full instructions and source code.
